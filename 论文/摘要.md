# 摘要

随着人工智能技术在智能交通与安防巡检领域的深入应用，传统的路径规划系统依赖人工配置结构化参数的方式已难以满足普通用户的自然交互需求，同时，复杂监控场景下单一视觉模型在多维度属性协同分析方面也存在不足。针对上述问题，本设计在有限的软硬件资源条件下，设计并实现了一个基于深度学习的智能导航与识别系统。

本系统采用前端富交互加轻量后端服务的混合架构。在路径规划方面，系统通过本地部署的大模型对自然语言出行需求进行语义解析，结合高德地图 API 完成多策略路径规划。在人物识别方面，系统融合目标检测模型与深度网络实现多属性联合识别，包括年龄估计、性别分类与服装颜色分析。在监控数据方面，系统集成多维度数据可视化面板与地理信息服务，支撑设备状态的实时监控与异常检测。在智能咨询方面，系统调用大模型生成咨询摘要并通过邮件服务完成自动化分发。此外，本系统基于 Vue3 与 FastAPI 设计了前后端分离的交互界面。经评估，本系统在智能导航、人物识别、监控数据可视化与智能咨询等功能上能够实现较好的预期效果。

**关键词**：深度学习；大规模语言模型；目标检测；路径规划；多模型融合；智能咨询
